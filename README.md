# Appfor-school

This is a simple calculator app built using Android Studio. It performs basic arithmetic operations including:

- Addition
- Subtraction
- Multiplication
- Division

## 💡 Features

- User-friendly interface
- Input validation
- Handles divide-by-zero errors
